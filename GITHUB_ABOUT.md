# GitHub About

## Short Description

Export opencode session history to a single global CSV so you can quickly see what each session was about and where it was opened.

## 中文描述

把 opencode 的 session 历史自动导出成一份全局 CSV，方便快速查看每个会话是做什么的、当时开在哪个目录。

## Suggested About Text

`Export opencode session history to a single global CSV with exact session directories, timestamps, and first prompts.`
